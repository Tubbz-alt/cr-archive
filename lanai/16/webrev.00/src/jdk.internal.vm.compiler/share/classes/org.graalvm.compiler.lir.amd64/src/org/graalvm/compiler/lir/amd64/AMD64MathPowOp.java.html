  59 /**
  60  * <pre>
  61  *                     ALGORITHM DESCRIPTION  - POW()
  62  *                     ---------------------
  63  *
  64  *    Let x=2^k * mx, mx in [1,2)
  65  *
  66  *    log2(x) calculation:
  67  *
  68  *    Get B~1/mx based on the output of rcpps instruction (B0)
  69  *    B = int((B0*LH*2^9+0.5))/2^9
  70  *    LH is a short approximation for log2(e)
  71  *
  72  *    Reduced argument, scaled by LH:
  73  *                r=B*mx-LH (computed accurately in high and low parts)
  74  *
  75  *    log2(x) result:  k - log2(B) + p(r)
  76  *             p(r) is a degree 8 polynomial
  77  *             -log2(B) read from data table (high, low parts)
  78  *             log2(x) is formed from high and low parts
  79  *    For |x| in [1-1/32, 1+1/16), a slower but more accurate computation
  80  *    based om the same table design is performed.
  81  *
  82  *   Main path is taken if | floor(log2(|log2(|x|)|) + floor(log2|y|) | < 8,
  83  *   to filter out all potential OF/UF cases.
  84  *   exp2(y*log2(x)) is computed using an 8-bit index table and a degree 5
  85  *   polynomial
  86  *
  87  * Special cases:
  88  *  pow(-0,y) = -INF and raises the divide-by-zero exception for y an odd
  89  *  integer < 0.
  90  *  pow(-0,y) = +INF and raises the divide-by-zero exception for y < 0 and
  91  *  not an odd integer.
  92  *  pow(-0,y) = -0 for y an odd integer > 0.
  93  *  pow(-0,y) = +0 for y > 0 and not an odd integer.
  94  *  pow(-1,-INF) = NaN.
  95  *  pow(+1,y) = NaN for any y, even a NaN.
  96  *  pow(x,-0) = 1 for any x, even a NaN.
  97  *  pow(x,y) = a NaN and raises the invalid exception for finite x < 0 and
  98  *  finite non-integer y.
  99  *  pow(x,-INF) = +INF for |x|<1.
 100  *  pow(x,-INF) = +0 for |x|>1.
 101  *  pow(x,+INF) = +0 for |x|<1.
 102  *  pow(x,+INF) = +INF for |x|>1.
 103  *  pow(-INF,y) = -0 for y an odd integer < 0.
 104  *  pow(-INF,y) = +0 for y < 0 and not an odd integer.
 105  *  pow(-INF,y) = -INF for y an odd integer > 0.
 106  *  pow(-INF,y) = +INF for y > 0 and not an odd integer.
 107  *  pow(+INF,y) = +0 for y <0.
 108  *  pow(+INF,y) = +INF for y >0.
 109  * </pre>
 110  */
