28 NOTE:
29 
30 JavaScript programs that are run in the context of a web page loaded by WebEngine can communicate with Java objects passed from the application to the JavaScript program. JavaScript programs that reference java.lang.Class objects are now limited to the following methods:
31 
32 ```
33 getCanonicalName
34 getEnumConstants
35 getFields
36 getMethods
37 getName
38 getPackageName
39 getSimpleName
40 getSuperclass
41 getTypeName
42 getTypeParameters
43 isAssignableFrom
44 isArray
45 isEnum
46 isInstance
47 isInterface
48 isLocalClass
49 isMemberClass
50 isPrimitive
51 isSynthetic
52 toGenericString
53 toString
54 ```
55 
56 No methods can be called on the following classes:
57 ```
58 java.lang.ClassLoader
59 java.lang.Module
60 java.lang.Runtime
61 java.lang.System
62 
63 java.lang.invoke.*
64 java.lang.module.*
65 java.lang.reflect.*
66 java.security.*
67 sun.misc.*
68 ```
