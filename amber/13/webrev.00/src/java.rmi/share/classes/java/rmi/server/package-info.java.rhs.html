26 /**
27  * Provides classes and interfaces for supporting the server side of RMI.
28  * One group of classes are used by the static stubs and skeletons.
29  * Another group of classes implements the RMI Transport protocol.





30  *
31  * <p><strong>Deprecated: Skeletons and Static Stubs.</strong>
32  *
33  * <em>Skeletons and statically generated stubs are deprecated.  This
34  * includes the APIs in this package that require the use of skeletons
35  * or static stubs and the runtime support for them.  Support for skeletons

36  * and static stubs may be removed in a future release of the
37  * platform. Skeletons are unnecessary, as server-side method dispatching
38  * is handled directly by the RMI runtime. Statically generated stubs are
39  * unnecessary, as stubs are generated dynamically using {@link
40  * java.lang.reflect.Proxy Proxy} objects. See {@link
41  * java.rmi.server.UnicastRemoteObject UnicastRemoteObject} for
42  * information about dynamic stub generation.</em>



43  *
44  * @since 1.1
45  */
