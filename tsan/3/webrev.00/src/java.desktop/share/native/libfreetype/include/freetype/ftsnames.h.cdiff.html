!   /*                                                                       */
!   /* <Note>                                                                */
!   /*    The `string' array returned in the `aname' structure is not        */

!   /*    by yourself; FreeType takes care of it if you call @FT_Done_Face.  */

!   /*    `name' table entries, then do a loop until you get the right       */
