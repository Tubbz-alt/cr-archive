 35 <h1>GIF Metadata Format Specification</h1>
 36 <a id="gif_stream_metadata_format"></a>
 37 <h2>GIF Stream Metadata Format Specification</h2>
 38 
 39 <p>
 40 
 41 The GIF stream metadata format encodes the information stored in the
 42 per-file header, namely the file version, logical screen descriptor,
 43 and the global color table.
 44 
 45 <p>
 46 
 47 The logical screen descriptor describes the overall width and height
 48 of an animated file, as well as information about the color resolution
 49 (number of bits per pixel) and pixel aspect ratio.
 50 
 51 <p>
 52 
 53 The global color table contains a color lookup table that is used for
 54 images that do not have their own local color table.
 55 
 56 <pre>
 57 &lt;!DOCTYPE "javax_imageio_gif_stream_1.0" [
 58 
 59   &lt;!ELEMENT "javax_imageio_gif_stream_1.0" (Version?,
 60     LogicalScreenDescriptor?, GlobalColorTable?)&gt;
 61 
 62     &lt;!ELEMENT "Version" EMPTY&gt;
 63       &lt;!-- The file version, either 87a or 89a --&gt;
 64       &lt;!ATTLIST "Version" "value" ("87a" | "89a") #REQUIRED&gt;
 65         &lt;!-- The version string --&gt;
 66 
 67     &lt;!ELEMENT "LogicalScreenDescriptor" EMPTY&gt;
 68       &lt;!-- The logical screen descriptor, except for the global color table --&gt;
 69       &lt;!ATTLIST "LogicalScreenDescriptor" "logicalScreenWidth" #CDATA
 70          #REQUIRED&gt;
 71         &lt;!-- The width in pixels of the whole picture --&gt;
 72         &lt;!-- Data type: Integer --&gt;
 73         &lt;!-- Min value: 1 (inclusive) --&gt;
 74         &lt;!-- Max value: 65535 (inclusive) --&gt;
 75       &lt;!ATTLIST "LogicalScreenDescriptor" "logicalScreenHeight" #CDATA
 76          #REQUIRED&gt;
 77         &lt;!-- The height in pixels of the whole picture --&gt;
 78         &lt;!-- Data type: Integer --&gt;
 79         &lt;!-- Min value: 1 (inclusive) --&gt;
 80         &lt;!-- Max value: 65535 (inclusive) --&gt;
 81       &lt;!ATTLIST "LogicalScreenDescriptor" "colorResolution" #CDATA
 82          #REQUIRED&gt;
 83         &lt;!-- The number of bits of color resolution, beteen 1 and 8 --&gt;
 84         &lt;!-- Data type: Integer --&gt;
 85         &lt;!-- Min value: 1 (inclusive) --&gt;
 86         &lt;!-- Max value: 8 (inclusive) --&gt;
 87       &lt;!ATTLIST "LogicalScreenDescriptor" "pixelAspectRatio" #CDATA
 88          #REQUIRED&gt;
 89         &lt;!-- If 0, indicates square pixels, else W/H = (value + 15)/64 --&gt;
 90         &lt;!-- Data type: Integer --&gt;
 91         &lt;!-- Min value: 0 (inclusive) --&gt;
 92         &lt;!-- Max value: 255 (inclusive) --&gt;
 93 
 94     &lt;!ELEMENT "GlobalColorTable" (ColorTableEntry)*&gt;
 95       &lt;!-- The global color table --&gt;
 96       &lt;!-- Min children: 2 --&gt;
 97       &lt;!-- Max children: 256 --&gt;
 98       &lt;!ATTLIST "GlobalColorTable" "sizeOfGlobalColorTable" ("2" |
 99         "4" | "8" | "16" | "32" | "64" | "128" | "256") #REQUIRED&gt;
100         &lt;!-- The number of entries in the global color table --&gt;
101       &lt;!ATTLIST "GlobalColorTable" "backgroundColorIndex" #CDATA
102          #REQUIRED&gt;
103         &lt;!-- The index of the color table entry to be used as a background --&gt;
104         &lt;!-- Data type: Integer --&gt;
105         &lt;!-- Min value: 0 (inclusive) --&gt;
106         &lt;!-- Max value: 255 (inclusive) --&gt;
107       &lt;!ATTLIST "GlobalColorTable" "sortFlag" ("TRUE" | "FALSE")
108          #REQUIRED&gt;
109         &lt;!-- True if the global color table is sorted by frequency --&gt;
110 
111       &lt;!ELEMENT "ColorTableEntry" EMPTY&gt;
112         &lt;!-- A global color table entry --&gt;
113         &lt;!ATTLIST "ColorTableEntry" "index" #CDATA #REQUIRED&gt;
114           &lt;!-- The index of the color table entry --&gt;
115           &lt;!-- Data type: Integer --&gt;
116           &lt;!-- Min value: 0 (inclusive) --&gt;
117           &lt;!-- Max value: 255 (inclusive) --&gt;
118         &lt;!ATTLIST "ColorTableEntry" "red" #CDATA #REQUIRED&gt;
119           &lt;!-- The red value for the color table entry --&gt;
120           &lt;!-- Data type: Integer --&gt;
121           &lt;!-- Min value: 0 (inclusive) --&gt;
122           &lt;!-- Max value: 255 (inclusive) --&gt;
123         &lt;!ATTLIST "ColorTableEntry" "green" #CDATA #REQUIRED&gt;
124           &lt;!-- The green value for the color table entry --&gt;
125           &lt;!-- Data type: Integer --&gt;
126           &lt;!-- Min value: 0 (inclusive) --&gt;
127           &lt;!-- Max value: 255 (inclusive) --&gt;
128         &lt;!ATTLIST "ColorTableEntry" "blue" #CDATA #REQUIRED&gt;
129           &lt;!-- The blue value for the color table entry --&gt;
130           &lt;!-- Data type: Integer --&gt;
131           &lt;!-- Min value: 0 (inclusive) --&gt;
132           &lt;!-- Max value: 255 (inclusive) --&gt;
133 ]&gt;

135 <a id="gif_image_metadata_format"></a>
136 <h2>GIF Image Metadata Format Specification</h2>
137 <p>
138 
139 The GIF image metadata format encodes the image descriptor, local
140 color table, and extension information assciated with a single image
141 within a GIF file, typically a frame of an animation.
142 
143 <p>
144 
145 The image description contains the offset of the frame within the
146 logical screen described in the stream metadata, as well as the width
147 and height of the frame and whether it is used interlaced encoding.
148 
149 <p>
150 
151 The local color table, if present, supersedes the global color table
152 for the frame at hand.
153 
154 <p>
155 
156 The graphic control extension contains information on how the frame is
157 to be incorporated into the animation.  The disposal method indicates
158 whether the current frame should remain in place
159 (<code>doNotDispose</code>), be restored to the backgound color as
160 specified in the stream metadata
161 (<code>restoreToBackgroundColor</code>), or be restored to the
162 previous frame contents (<code>restoreToPrevious</code>) prior to
163 displaying the subsequent frame.
164 
165 <p>
166 
167 The user input flag indicates whether the animation should pause for
168 user input before advancing to the next frame.  The transparent color
169 flag and index indicate a color index for which pixels of the frame
170 should not be drawn.  The delay time indicates how long the frame
171 should be displayed, in hundredths of a second.  If the user input
172 flag is set and the delay time is 0, the frame is intended to be
173 advanced only on user input.
174 
175 <pre>
176 &lt;!DOCTYPE "javax_imageio_gif_image_1.0" [
177 
178   &lt;!ELEMENT "javax_imageio_gif_image_1.0" (ImageDescriptor?,
179     LocalColorTable?, GraphicControlExtension?, PlainTextExtension?,
180     ApplicationExtensions?, CommentExtensions?)&gt;
181 
182     &lt;!ELEMENT "ImageDescriptor" EMPTY&gt;
183       &lt;!-- The image descriptor --&gt;
184       &lt;!ATTLIST "ImageDescriptor" "imageLeftPosition" #CDATA #REQUIRED&gt;
185         &lt;!-- The X offset of the image relative to the screen origin --&gt;
186         &lt;!-- Data type: Integer --&gt;
187         &lt;!-- Min value: 0 (inclusive) --&gt;
188         &lt;!-- Max value: 65535 (inclusive) --&gt;
189       &lt;!ATTLIST "ImageDescriptor" "imageTopPosition" #CDATA #REQUIRED&gt;
190         &lt;!-- The Y offset of the image relative to the screen origin --&gt;
191         &lt;!-- Data type: Integer --&gt;
192         &lt;!-- Min value: 0 (inclusive) --&gt;
193         &lt;!-- Max value: 65535 (inclusive) --&gt;
194       &lt;!ATTLIST "ImageDescriptor" "imageWidth" #CDATA #REQUIRED&gt;
195         &lt;!-- The width of the image --&gt;
196         &lt;!-- Data type: Integer --&gt;
197         &lt;!-- Min value: 1 (inclusive) --&gt;
198         &lt;!-- Max value: 65535 (inclusive) --&gt;
199       &lt;!ATTLIST "ImageDescriptor" "imageHeight" #CDATA #REQUIRED&gt;
200         &lt;!-- The height of the image --&gt;
201         &lt;!-- Data type: Integer --&gt;
202         &lt;!-- Min value: 1 (inclusive) --&gt;
203         &lt;!-- Max value: 65535 (inclusive) --&gt;
204       &lt;!ATTLIST "ImageDescriptor" "interlaceFlag" ("TRUE" | "FALSE")
205          #REQUIRED&gt;
206         &lt;!-- True if the image is stored using interlacing --&gt;
207 
208     &lt;!ELEMENT "LocalColorTable" (ColorTableEntry)*&gt;
209       &lt;!-- The local color table --&gt;
210       &lt;!-- Min children: 2 --&gt;
211       &lt;!-- Max children: 256 --&gt;
212       &lt;!ATTLIST "LocalColorTable" "sizeOfLocalColorTable" ("2" |
213         "4" | "8" | "16" | "32" | "64" | "128" | "256") #REQUIRED&gt;
214         &lt;!-- The number of entries in the local color table --&gt;
215       &lt;!ATTLIST "LocalColorTable" "sortFlag" ("TRUE" | "FALSE")
216          #REQUIRED&gt;
217         &lt;!-- True if the local color table is sorted by frequency --&gt;
218 
219       &lt;!ELEMENT "ColorTableEntry" EMPTY&gt;
220         &lt;!-- A local color table entry --&gt;
221         &lt;!ATTLIST "ColorTableEntry" "index" #CDATA #REQUIRED&gt;
222           &lt;!-- The index of the color table entry --&gt;
223           &lt;!-- Data type: Integer --&gt;
224           &lt;!-- Min value: 0 (inclusive) --&gt;
225           &lt;!-- Max value: 255 (inclusive) --&gt;
226         &lt;!ATTLIST "ColorTableEntry" "red" #CDATA #REQUIRED&gt;
227           &lt;!-- The red value for the color table entry --&gt;
228           &lt;!-- Data type: Integer --&gt;
229           &lt;!-- Min value: 0 (inclusive) --&gt;
230           &lt;!-- Max value: 255 (inclusive) --&gt;
231         &lt;!ATTLIST "ColorTableEntry" "green" #CDATA #REQUIRED&gt;
232           &lt;!-- The green value for the color table entry --&gt;
233           &lt;!-- Data type: Integer --&gt;
234           &lt;!-- Min value: 0 (inclusive) --&gt;
235           &lt;!-- Max value: 255 (inclusive) --&gt;
236         &lt;!ATTLIST "ColorTableEntry" "blue" #CDATA #REQUIRED&gt;
237           &lt;!-- The blue value for the color table entry --&gt;
238           &lt;!-- Data type: Integer --&gt;
239           &lt;!-- Min value: 0 (inclusive) --&gt;
240           &lt;!-- Max value: 255 (inclusive) --&gt;
241 
242     &lt;!ELEMENT "GraphicControlExtension" EMPTY&gt;
243       &lt;!-- A graphic control extension --&gt;
244       &lt;!ATTLIST "GraphicControlExtension" "disposalMethod" ("none" |
245         "doNotDispose" | "restoreToBackgroundColor" |
246         "restoreToPrevious" | "undefinedDisposalMethod4" |
247         "undefinedDisposalMethod5" | "undefinedDisposalMethod6" |
248         "undefinedDisposalMethod7") #REQUIRED&gt;
249         &lt;!-- The disposal method for this frame --&gt;
250       &lt;!ATTLIST "GraphicControlExtension" "userInputFlag" ("TRUE" |
251         "FALSE") #REQUIRED&gt;
252         &lt;!-- True if the frame should be advanced based on user input --&gt;
253       &lt;!ATTLIST "GraphicControlExtension" "transparentColorFlag" (
254         "TRUE" | "FALSE") #REQUIRED&gt;
255         &lt;!-- True if a transparent color exists --&gt;
256       &lt;!ATTLIST "GraphicControlExtension" "delayTime" #CDATA #REQUIRED&gt;
257         &lt;!-- The time to delay between frames, in hundredths of a second --&gt;
258         &lt;!-- Data type: Integer --&gt;
259         &lt;!-- Min value: 0 (inclusive) --&gt;
260         &lt;!-- Max value: 65535 (inclusive) --&gt;
261       &lt;!ATTLIST "GraphicControlExtension" "transparentColorIndex"
262          #CDATA #REQUIRED&gt;
263         &lt;!-- The transparent color, if transparentColorFlag is true --&gt;
264         &lt;!-- Data type: Integer --&gt;
265         &lt;!-- Min value: 0 (inclusive) --&gt;
266         &lt;!-- Max value: 255 (inclusive) --&gt;
267 
268     &lt;!ELEMENT "PlainTextExtension" EMPTY&gt;
269       &lt;!-- A plain text (text grid) extension --&gt;
270       &lt;!ATTLIST "PlainTextExtension" "textGridLeft" #CDATA #REQUIRED&gt;
271         &lt;!-- The X offset of the text grid --&gt;
272         &lt;!-- Data type: Integer --&gt;
273         &lt;!-- Min value: 0 (inclusive) --&gt;
274         &lt;!-- Max value: 65535 (inclusive) --&gt;
275       &lt;!ATTLIST "PlainTextExtension" "textGridTop" #CDATA #REQUIRED&gt;
276         &lt;!-- The Y offset of the text grid --&gt;
277         &lt;!-- Data type: Integer --&gt;
278         &lt;!-- Min value: 0 (inclusive) --&gt;
279         &lt;!-- Max value: 65535 (inclusive) --&gt;
280       &lt;!ATTLIST "PlainTextExtension" "textGridWidth" #CDATA #REQUIRED&gt;
281         &lt;!-- The number of columns in the text grid --&gt;
282         &lt;!-- Data type: Integer --&gt;
283         &lt;!-- Min value: 1 (inclusive) --&gt;
284         &lt;!-- Max value: 65535 (inclusive) --&gt;
285       &lt;!ATTLIST "PlainTextExtension" "textGridHeight" #CDATA #REQUIRED&gt;
286         &lt;!-- The number of rows in the text grid --&gt;
287         &lt;!-- Data type: Integer --&gt;
288         &lt;!-- Min value: 1 (inclusive) --&gt;
289         &lt;!-- Max value: 65535 (inclusive) --&gt;
290       &lt;!ATTLIST "PlainTextExtension" "characterCellWidth" #CDATA
291          #REQUIRED&gt;
292         &lt;!-- The width of a character cell --&gt;
293         &lt;!-- Data type: Integer --&gt;
294         &lt;!-- Min value: 1 (inclusive) --&gt;
295         &lt;!-- Max value: 255 (inclusive) --&gt;
296       &lt;!ATTLIST "PlainTextExtension" "characterCellHeight" #CDATA
297          #REQUIRED&gt;
298         &lt;!-- The height of a character cell --&gt;
299         &lt;!-- Data type: Integer --&gt;
300         &lt;!-- Min value: 1 (inclusive) --&gt;
301         &lt;!-- Max value: 255 (inclusive) --&gt;
302       &lt;!ATTLIST "PlainTextExtension" "textForegroundColor" #CDATA
303          #REQUIRED&gt;
304         &lt;!-- The text foreground color index --&gt;
305         &lt;!-- Data type: Integer --&gt;
306         &lt;!-- Min value: 0 (inclusive) --&gt;
307         &lt;!-- Max value: 255 (inclusive) --&gt;
308       &lt;!ATTLIST "PlainTextExtension" "textBackgroundColor" #CDATA
309          #REQUIRED&gt;
310         &lt;!-- The text background color index --&gt;
311         &lt;!-- Data type: Integer --&gt;
312         &lt;!-- Min value: 0 (inclusive) --&gt;
313         &lt;!-- Max value: 255 (inclusive) --&gt;
314 
315     &lt;!ELEMENT "ApplicationExtensions" (ApplicationExtension)*&gt;
316       &lt;!-- A set of application extensions --&gt;
317       &lt;!-- Min children: 1 --&gt;
318       &lt;!-- Max children: 2147483647 --&gt;
319 
320       &lt;!ELEMENT "ApplicationExtension" EMPTY&gt;
321         &lt;!-- An application extension --&gt;
322         &lt;!-- User object: array of byte --&gt;
323         &lt;!-- Min length: 0 --&gt;
324         &lt;!-- Max length: 2147483647 --&gt;
325         &lt;!ATTLIST "ApplicationExtension" "applicationID" #CDATA
326            #REQUIRED&gt;
327           &lt;!-- The application ID --&gt;
328           &lt;!-- Data type: String --&gt;
329         &lt;!ATTLIST "ApplicationExtension" "authenticationCode" #CDATA
330            #REQUIRED&gt;
331           &lt;!-- The authentication code --&gt;
332           &lt;!-- Data type: String --&gt;
333 
334     &lt;!ELEMENT "CommentExtensions" (CommentExtension)*&gt;
335       &lt;!-- A set of comments --&gt;
336       &lt;!-- Min children: 1 --&gt;
337       &lt;!-- Max children: 2147483647 --&gt;
338 
339       &lt;!ELEMENT "CommentExtension" EMPTY&gt;
340         &lt;!-- A comment --&gt;
341         &lt;!ATTLIST "CommentExtension" "value" #CDATA #REQUIRED&gt;
342           &lt;!-- The comment --&gt;
343           &lt;!-- Data type: String --&gt;
344 ]&gt;
