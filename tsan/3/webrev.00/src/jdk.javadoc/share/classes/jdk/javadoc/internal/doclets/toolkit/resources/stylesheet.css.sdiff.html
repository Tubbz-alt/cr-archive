810 /*
811  * Styles for user-provided tables.
812  *
813  * borderless:
814  *      No borders, vertical margins, styled caption.
815  *      This style is provided for use with existing doc comments.
816  *      In general, borderless tables should not be used for layout purposes.
817  *
818  * plain:
819  *      Plain borders around table and cells, vertical margins, styled caption.
820  *      Best for small tables or for complex tables for tables with cells that span
821  *      rows and columns, when the "striped" style does not work well.
