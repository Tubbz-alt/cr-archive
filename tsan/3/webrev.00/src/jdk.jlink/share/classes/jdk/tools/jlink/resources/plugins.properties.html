 46 compress.argument=<0|1|2>[:filter=<pattern-list>]
 47 
 48 compress.description=\
 49 Compress all resources in the output image.\n\
 50 Level 0: No compression\n\
 51 Level 1: Constant string sharing\n\
 52 Level 2: ZIP.\n\
 53 An optional <pattern-list> filter can be specified to list the pattern of\n\
 54 files to be included.
