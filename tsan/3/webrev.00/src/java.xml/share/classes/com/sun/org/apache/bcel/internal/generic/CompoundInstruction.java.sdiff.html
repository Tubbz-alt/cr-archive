27  * the (virtual) PUSH instruction takes an arbitray argument and produces the

29  * SWITCH instruction as a useful template for either LOOKUPSWITCH or
30  * TABLESWITCH.

34  *
35  * @version $Id: CompoundInstruction.java 1747278 2016-06-07 17:28:43Z britter $
36  * @see PUSH
37  * @see SWITCH
