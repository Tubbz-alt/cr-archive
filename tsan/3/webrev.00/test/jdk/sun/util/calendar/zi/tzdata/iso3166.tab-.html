 27 # From Paul Eggert (2015-05-02):
 28 # This file contains a table of two-letter country codes.  Columns are
 29 # separated by a single tab.  Lines beginning with '#' are comments.
 30 # All text uses UTF-8 encoding.  The columns of the table are as follows:
 31 #
 32 # 1.  ISO 3166-1 alpha-2 country code, current as of
 33 #     ISO 3166-1 N905 (2016-11-15).  See: Updates on ISO 3166-1
 34 #     http://isotc.iso.org/livelink/livelink/Open/16944257
 35 # 2.  The usual English name for the coded region,
 36 #     chosen so that alphabetic sorting of subsets produces helpful lists.
 37 #     This is not the same as the English name in the ISO 3166 tables.
 38 #
 39 # The table is sorted by country code.
 40 #
 41 # This table is intended as an aid for users, to help them select time
 42 # zone data appropriate for their practical needs.  It is not intended
 43 # to take or endorse any position on legal or territorial claims.
