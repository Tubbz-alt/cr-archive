35 # The International Earth Rotation and Reference Systems Service
36 # periodically uses leap seconds to keep UTC to within 0.9 s of UT1
37 # (which measures the true angular orientation of the earth in space)
38 # and publishes leap second data in a copyrighted file
39 # <https://hpiers.obspm.fr/iers/bul/bulc/Leap_Second.dat>.
40 # See: Levine J. Coordinated Universal Time and the leap second.
41 # URSI Radio Sci Bull. 2016;89(4):30-6. doi:10.23919/URSIRSB.2016.7909995
42 # <https://ieeexplore.ieee.org/document/7909995>.
43 # There were no leap seconds before 1972, because the official mechanism
44 # accounting for the discrepancy between atomic time and the earth's rotation
45 # did not exist.
46 
47 # The correction (+ or -) is made at the given time, so lines
48 # will typically look like:
49 #	Leap	YEAR	MON	DAY	23:59:60	+	R/S
50 # or
51 #	Leap	YEAR	MON	DAY	23:59:59	-	R/S
52
