 40 main.opt.public.desc=\
 41     Show only public types and members. For named modules,\n\
 42     show exported packages and the module''s API.
 43 
 44 main.opt.protected.desc=\
 45     Show protected/public types and members (default). For\n\
 46     named modules, show exported packages and the module''s API.
 47 
 48 main.opt.package.desc=\
 49     Show package/protected/public types and members. For \n\
 50     named modules, show all packages and all module details.
 51 
 52 main.opt.private.desc=\
 53     Show all types and members. For named modules,\n\
 54     show all packages and all module details.
 55 
 56 main.opt.show.members.arg=\
 57     <value>
 58 main.opt.show.members.desc=\
 59     Specifies which members (fields, methods, etc.) will be\n\
 60     documented, where value can be one of "public", "protected",\n\
 61     "package" or "private". The default is "protected", which will\n\
 62     show public and protected members, "public" will show only\n\
 63     public members, "package" will show public, protected and\n\
 64     package members and "private" will show all members.
 65 
 66 main.opt.show.types.arg=\
 67     <value>
 68 main.opt.show.types.desc=\
 69     Specifies which types (classes, interfaces, etc.) will be\n\
 70     documented, where value can be one of "public", "protected",\n\
 71     "package" or "private". The default is "protected", which will\n\
 72     show public and protected types, "public" will show only\n\
 73     public types, "package" will show public, protected and\n\
 74     package types and "private" will show all types.
 75 
 76 main.opt.show.packages.arg=\
 77     <value>
 78 main.opt.show.packages.desc=\
 79     Specifies which module's packages will be documented. Possible\n\
 80     values are "exported" or "all" packages.
 81 
 82 main.opt.show.module.contents.arg=\
 83     <value>
 84 main.opt.show.module.contents.desc=\
 85     Specifies the documentation granularity of module\n\
 86     declarations. Possible values are "api" or "all".
 87 
 88 main.opt.expand.requires.arg=\
 89     <value>
 90 main.opt.expand.requires.desc=\
 91     Instructs the tool to expand the set of modules to be\n\
 92     documented. By default, only the modules given explicitly on\n\
 93     the command line will be documented. A value of "transitive"\n\
 94     will additionally include all "requires transitive"\n\
 95     dependencies of those modules. A value of "all" will include\n\
 96     all dependencies of those modules.
