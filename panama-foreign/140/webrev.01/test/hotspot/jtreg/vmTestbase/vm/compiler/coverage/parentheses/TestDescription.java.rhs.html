27  * @key randomness
28  * @modules java.base/jdk.internal.org.objectweb.asm:+open java.base/jdk.internal.misc:+open
29  *
30  * @summary converted from VM Testbase vm/compiler/coverage/parentheses.
31  * VM Testbase keywords: [quick, stressopt, jit]
32  * VM Testbase readme:
33  * DESCRIPTION
34  *     This test works as follows:
35  *     1. Generates random proper list of following JVM instructions:
36  *     -Integer constants: ICONST_M1, ICONST_0, ICONST_1, ICONST_2, ICONST_3, ICONST_4, ICONST_5
37  *     -Integer arithmetics: IADD, ISUB, IMUL, IAND, IOR, IXOR, ISHL, ISHR, INEG
38  *     -Stack operations: DUP, SWAP
39  *     -NOP
40  *     (If loadFrom command line option is setted, test will be use instruction sequence from file.
41  *     It is useful for reproducing failures.)
42  *     2. Executes this sequence in to ways: with TinyInstructionsExecutor and  HotspotInstructionsExecutor.
43  *     3. Checks that results of execution are equals.
44  *     For generation instructions test using right parentheses sequence. Open parenthesis "(" is correspondent to
45  *     stack increasing instructions, ")" is corresponded stack decreasing instructions.
46  *
47  * @library /vmTestbase
48  *          /test/lib
49  * @run driver jdk.test.lib.FileInstaller . .
50  * @run main/othervm vm.compiler.coverage.parentheses.Parentheses
